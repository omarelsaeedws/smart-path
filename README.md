 Smart Path 🚀

Smart Path is an AI-powered web application designed to help students build personalized learning paths based on their goals, current level, and available study time.

 📌 Overview

With the rapid growth of online learning resources, students often face difficulty choosing the right path and organizing their learning journey. Smart Path solves this problem by providing a structured, intelligent, and personalized learning experience.

 🤖 AI Feature

The system uses AI to analyze user inputs such as:

* Learning goal
* Current skill level
* Available study time

Based on this data, it generates a customized learning roadmap divided into levels (Beginner → Intermediate → Advanced), with recommended courses and resources. The roadmap updates dynamically when user data changes.

## 🧠 Quiz System

Each lesson includes an interactive quiz designed to:

* Evaluate the user’s understanding
* Reinforce learning outcomes
* Ensure progression to the next level based on performance

This feature enhances engagement and makes the learning process more effective and measurable.

## ⚙️ Features

* AI-generated personalized learning paths
* Smart course and resource recommendations
* Interactive quizzes for each lesson
* Progress tracking system
* Dynamic roadmap updates
* Admin panel for managing users and content

## 🛠️ Tech Stack

* Frontend: React.js + Tailwind CSS
* Backend: Firebase (Auth, Firestore, Cloud Functions)
* AI Integration: Adaptive roadmap generation

## 🎯 Goal

The project aims to improve self-learning efficiency by organizing learning paths, reducing confusion, and helping users gain real-world skills aligned with market needs.
