# Smart Path 🚀

Smart Path is an AI-powered web application designed to help students build personalized learning paths based on their goals, current level, and available study time.

## 📌 Overview

With the rapid growth of online learning resources, students often face difficulty choosing the right path and organizing their learning journey. Smart Path solves this problem by providing a structured, intelligent, and personalized learning experience.

## 🤖 AI Feature

The core of the system is an AI module that analyzes user inputs such as:

* Learning goal
* Current skill level
* Available study time

Based on this data, the AI generates a customized learning roadmap divided into multiple levels (Beginner → Intermediate → Advanced), along with recommended courses and tools for each stage. The system can also regenerate the roadmap dynamically when user data changes.

## ⚙️ Features

* Personalized learning paths using AI
* Smart recommendations for courses and resources
* Progress tracking system
* Dynamic roadmap updates
* Admin panel for managing users and content

## 🛠️ Tech Stack

* Frontend: React.js + Tailwind CSS
* Backend: Firebase (Auth, Firestore, Cloud Functions)
* AI Integration: Generates adaptive learning roadmaps

## 🎯 Goal

The project aims to enhance self-learning efficiency by reducing confusion, organizing learning processes, and helping students acquire skills aligned with job market needs.
